# Self-Hosted GitHub Actions Runners documentation

This documentation is available on https://canonical-self-hosted-github-runner-docs.readthedocs-hosted.com/

## Structure

This section outlines the structure of this repository, and some key files.

### `docs/`

This directory contains the documentation sources.

To view it in your browser, navigate to this directory and type `make run`.

### `.github/workflows/`

This directory contains files used for documentation build checks via GitHub's CI.
