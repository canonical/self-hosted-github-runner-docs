# How can I increase the time limit for my job?

The default limit for a job (a workflow may be composed into multiple jobs using multiple runners) is usually 6 hours, but this limit can be increased using the [timeout-minutes](https://docs.github.com/en/actions/writing-workflows/workflow-syntax-for-github-actions#jobsjob_idtimeout-minutes) key up to 5 days.
